把weixin这个文件夹放到电脑桌面
修改exe.bat中路径为自己桌面

即下面2行中的C:\Users\ZY\Desktop/weixin替换为自己桌面
adb pull /sdcard/picname.png C:\Users\ZY\Desktop\weixin
cd C:\Users\ZY\Desktop\weixin

下载后创建wexin文件夹，把下载东西全拷过去，或者去Main21代码里面修改路径
把weixin这个文件夹放到电脑桌面
打开微信，进入如图picname类似的界面，然后点击exe.bat
测试手机分辨率：1080*1920
